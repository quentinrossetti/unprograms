Functions

dec_to_rgb (c_dec)
Convert a RGB color in decimal reprensentation into it's 3 values (red, green and blue)

Parameters:

  • c_dec number RGB color as a decimal reprensentation.

Returns:

  1. number Red value [0, 255]
  2. number Green value [0, 255]
  3. number Blue value [0, 255]

Usage:

    dec_to_rgb(0xb3310)
--> 11      51      16
distance (c1_dec, c2_dec)
Calculate the distance between two colors in RGB space using the low-cost approximation without gamma correction. Algorithm based on CompuPhase article

Parameters:

  • c1_dec number Color 1 as a decimal reprensentation.
  • c2_dec number Color 2 as a decimal reprensentation.

Returns:

    number Distance between color A and B. The max distance is 442.40365741453

Usage:

    distance(0xb3310, 0x174bff)
--> 412.24779221665
dec_to_ansi (c_dec)
Convert a RGB color in decimal reprensentation into it's clostest ANSI code Colors based on xterm. Algorithm based on CompuPhase article.

Parameters:

  • c_dec number RGB color as a decimal reprensentation.

Returns:

    number Clostest ANSI color code.

Usage:

    local ansi_foreground = "\27["..tostring(dec_to_ansi(0x174bff)).."m"
local ansi_background = "\27["..tostring(dec_to_ansi(0x174bff)+10).."m"
print("\27[0m") -- reset
