Functions

it (title, f[, timeout])
Create a test case. A test case should contain one ore more assert. A test fails when error is called. This function is global, you can call it from anywhere.

Parameters:

  • title string Description of what is being tested
  • f function Function that contains the code or your test
  • timeout number Max amount of seconds to wait for this test. (optional)

Usage:

    it("sould return a value between 0 and 100", function ()
  local percent = my_program.getEnergyPercentage()
  assert(percent >= 0)
  assert(percent <= 100)
end)
describe (title, f)
Group tests. Using describe() create a suite which contains all the tests under it. This function is global, you can call it from anywhere.

Parameters:

  • title string Description of the suite
  • f function Function that contains individual tests created using the it() function.

Usage:

    decribe("Tests about the energy feature", function ()
  it("should charge battery when under 10%", function ()
    -- some test
  end)
  it("should stop charging battery when above 80%", function ()
    -- some test
  end)
  it("should return a number between 0% and 100%", function ()
    -- some test
  end)
end)
before_each ()
Runs some shared setup before each test

Usage:

  -- do stuff before each test
  -- e.g: move items back to chest, reset some condition, etc.
end
after_each ()

Usage:

  -- do stuff after each test
end

Tables

context
Stores infos about the current test run.

Fields:

  • timeout number Max amount of seconds to wait for each test.
  • trace boolean If a traceback has to be displayed

Usage:

    context.timeout = 4 -- wait 4 seconds before stopping a test.
context.trace = true -- print the traceback of errors
